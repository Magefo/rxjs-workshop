In this demo we use 2 different ways to listen to the events of a button click.

Both button clicks will log the a message to the 'Message' panel below, and log the raw output in the browsers console. Make sure to open the console to see this data.

The first button uses the default addEventListener method provided by the browser.

The second button uses the fromEvent Observable.
